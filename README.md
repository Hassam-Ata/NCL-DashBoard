# DashBoard
This is a dashboard which fetches the data from the API and displays the data in tabular format. It also provide search and filter functionality. 
## To run the project:
clone the project using :
``` bash
  git clone "URL of the repo"
```
go to client folder 
``` bash
cd client
```
run 
``` bash
npm i 
```
then run:
``` bash
npm run dev
```
